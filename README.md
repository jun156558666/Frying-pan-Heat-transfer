# フライパン熱伝導 FDM シミュレーション

T-fal 24cm フライパンを対象とした非定常熱伝導の数値シミュレーション。  
**2 種類のモデル**（平板モデル・シェルモデル）で 5 種類の材料を比較し、SolidWorks 3D 熱解析との精度を評価する。

| スクリプト | モデル | 概要 |
|---|---|---|
| `cooling_simulation.py` | 平板モデル（2D FDM） | 底面を 2D 格子で離散化、側壁冷却を等価対流係数に集約 |
| `cooling_simulation_bowl.py` | シェルモデル（1D Shell） | 底面→コーナー→側壁を中立面弧長方向の 1D で統合 |

| フェーズ | 時間 | バーナー |
|---|---|---|
| 加熱 | t = 0 〜 300 s | ON（q_s = 15,000 W/m²） |
| 冷却 | t = 300 〜 600 s | OFF（自然対流のみ） |

| 出力ファイル | スクリプト | 内容 |
|---|---|---|
| `top_view_300s.png` | 平板 | 上面視カラーマップ t = 300 s |
| `top_view_600s.png` | 平板 | 上面視カラーマップ t = 600 s |
| `bowl_top_view_300s.png` | シェル | 上面視カラーマップ t = 300 s |
| `bowl_top_view_600s.png` | シェル | 上面視カラーマップ t = 600 s |

## 解析モデル①：平板モデル（`cooling_simulation.py`）

```
        ← L=122mm (R=61mm) →← L_h=207mm →
        ┌────────────────────┬────────────┐
        │   円形底面 (2D)    │  持ち手    │
        │  FDM (Ny×Nx 格子) │  (1D フィン)│
        └────────────────────┴────────────┘
        ↑ バーナー加熱（中央 R_b=90.3mm）
```

- **底面**：2次元 Backward Euler FDM（122×31 格子）
- **持ち手**：1次元フィン方程式（Backward Euler、104 節点）
- 側壁冷却を底面外縁の等価対流係数に集約（簡略化）

## 解析モデル②：シェルモデル（`cooling_simulation_bowl.py`）

```
        底面 (r=0→110mm)
           ↓
        コーナー（直線近似、3→2mm テーパー）
           ↓
        側壁 (r=121mm, h=47mm)
           ↓
        持ち手（1D フィン、根元=側壁上縁）
```

- **シェル弧長方向 1D**（169 節点）でボウル形状を統合表現
- 中心軸 r=0 に L'Hôpital 極限を適用
- 持ち手根元は側壁最上縁ノード温度を Dirichlet BC として接続

## 数値解法

各タイムステップで連立1次方程式を解く：

$$\frac{T^{n+1} - T^n}{\Delta t} = \mathcal{L}(T^{n+1}) + f$$

行列は LU 分解（`scipy.sparse.linalg.factorized`）で前処理し、各ステップの求解コストを最小化している。

**シェルモデルの支配方程式（円筒座標）：**

$$\rho c \, t \, \frac{\partial T}{\partial t} = \frac{1}{r} \frac{\partial}{\partial s}\!\left(k \, t \, r \, \frac{\partial T}{\partial s}\right) + h_o(T_o - T) + h_i(T_i - T) + q_s$$

中心軸 r=0（i=0）は L'Hôpital 極限により：

$$A_{0,0} = \frac{\rho c \, t_0}{\Delta t} + \frac{4 k \, t_0}{\Delta s^2} + h_\text{conv}, \quad A_{0,1} = -\frac{4 k \, t_0}{\Delta s^2}$$

## シミュレーション結果（t = 300 s）──空焚き・加熱終了時

> **最高温度** = 底面外面（バーナー側）補正後の全節点最大値

| フェノール樹脂 (PF) | 477.6 | 520.8 | +9.0% | 18.9 | 20.0 | +6% |

### シェルモデル vs SolidWorks

| 素材 | SW 最高 [°C] | SIM 最高 [°C] | 誤差 |
|---|---:|---:|---:|
| 銅 | 141.7 | 134.2 | −5.3% |
| アルミ (A5052P) | 207.5 | 171.8 | −17.2% |
| 鉄（ねずみ鋳鉄） | 251.2 | 213.0 | −15.2% |
| ステンレス (SUS304) | 294.3 | 248.9 | −15.4% |
| フェノール樹脂 (PF) | 477.6 | 397.6 | −16.8% |

**考察（t = 300 s）**

- 平板モデルは全素材で SolidWorks より**高め**（過大評価）
- シェルモデルは全素材で SolidWorks より**低め**（過小評価）
- → SolidWorks の値は両モデルの間に位置する
- 銅の平板誤差（+21.7%）：側壁からの大きな冷却が簡略化で不足するため
- シェルモデルの銅は −5.3% と比較的良好

### シェルモデル vs SolidWorks

| 素材 | SW 最高 [°C] | SIM 最高 [°C] | 誤差 | SW 最低 [°C] | SIM 最低 [°C] |
|---|---:|---:|---:|---:|---:|
| 銅 | 51.8 | 51.9 | +0.1% | 48.9 | 51.9 |
| アルミ (A5052P) | 48.8 | 49.4 | +1.2% | 38.8 | 49.4 |
| 鉄（ねずみ鋳鉄） | 88.5 | 53.6 | −39.4% | 20.3 | 23.7 |
| ステンレス (SUS304) | 146.7 | 82.2 | −44.0% | 18.9 | 20.0 |
| フェノール樹脂 (PF) | 246.1 | 84.8 | −65.5% | 18.9 | 20.0 |

**考察（t = 600 s）── モデルの棲み分け**

| | 低 k 素材（鉄・SUS・PF） | 高 k 素材（銅・アルミ） |
|---|---|---|
| 平板モデル | **±4% 以内**（高精度） | +18〜32%（過大） |
| シェルモデル | −40〜65%（大幅過小） | **±1% 台**（超高精度） |

- **平板モデル** は冷却期の低 k 素材に強い：熱が外に逃げにくく側壁冷却の影響が小さいため
- **シェルモデル** は冷却期の高 k 素材に強い：等温化が速くボウル形状の幾何学的表現が有効
- 2 モデルは素材の熱拡散率に応じて「棲み分け」する補完的な関係にある
